## Overview

Mforth support for the [Sublime Text](https://sublimetext.com) editor.

## Usage

Symlink this `./sublime` directory to your Sublime packages directory. Example for MacOS:

```sh
git clone https://github.com/mitranim/mforth.git
cd mforth
ln -sf "$(pwd)/sublime" "$HOME/Library/Application Support/Sublime Text/Packages/mforth"
```

To find the packages directory on your system, use Sublime Text menu → Preferences → Browse Packages.
